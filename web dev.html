What is web development?

Web devoplment refers to developing, deploying & maintaining the websites for the different purposes like
business, education and management etc. The website inculdes mainly two parts.

Parts of web development

  2. Backend Development

Backend Development

Backend development refers to that part of website that is not visible to us. It is used to insert, retrieve, delete & update
the data to the database for example when we create facebook account then we need to fill our moblie number/ Emial Id for login.
It all includes our backend part.
